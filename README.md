# CreateLyricMarker

---

## is何

* After Effectsでvsqxを読み込んで、ノートの位置にマーカーを置いたテキストレイヤーを作成します
* 全部のトラック分作成します

## How to use

* After Effectsを起動、コンポジションを作成 or 開きます
    * 対象となるコンポジションのタブを選択しておいてください
* ファイル -> スクリプト -> スクリプトファイルを実行... からCreateLyricMarker.jsxを選択
* ファイル選択ダイアログが出るのでvsqxを選択
* 完了
